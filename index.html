แบบประเมินความเหมาะสมของแบบจำลอง Machine Learning และ Dashboard

การพัฒนาแบบจำลอง Machine Learning เพื่อวิเคราะห์ความสัมพันธ์ระหว่างผลสัมฤทธิ์ทางการเรียนในห้องเรียนกับผลการสอบมาตรฐาน Ordinary National Educational Test ของนักเรียนชั้นมัธยมศึกษา
คำชี้แจง:แบบประเมินฉบับนี้มีวัตถุประสงค์เพื่อสอบถามความคิดเห็นของผู้เชี่ยวชาญเกี่ยวกับความเหมาะสมขององค์ประกอบต่าง ๆ ในการพัฒนาระบบ โดยแบบประเมินแบ่งออกเป็น 3 ส่วน ประกอบด้วย

ด้านที่ 3: ความเหมาะสมของการออกแบบ Dashboard
รายการประเมิน 5 4 3 2 1
3.1 การแสดงผล "แนวโน้มการเปลี่ยนแปลงคะแนน" มีความชัดเจนและสื่อความหมายได้ดี
3.2 การแสดงผล "การเปรียบเทียบความสัมพันธ์" ระหว่างคะแนนในห้องเรียนและ O-NET มีประโยชน์ต่อการวิเคราะห์
3.3 การแสดงผล "การจัดอันดับรายวิชา" ที่มีความสัมพันธ์สูงกับ O-NET เป็นประโยชน์ต่อการวางแผนนโยบาย
3.4 การแสดงผล "กลุ่มเสี่ยง" มีความชัดเจนและช่วยให้สามารถระบุกลุ่มเป้าหมายได้
3.5 การแสดงผล "ค่าความแม่นยำของโมเดล" ช่วยสร้างความน่าเชื่อถือให้กับข้อมูลที่นำเสนอ
3.6 องค์ประกอบโดยรวมของ Dashboard สามารถสื่อสารข้อมูลเชิงลึกแก่ผู้บริหารได้อย่างมีประสิทธิภาพ
